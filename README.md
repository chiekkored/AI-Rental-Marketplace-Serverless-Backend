# AI Rental Marketplace Serverless Backend

A Firebase Cloud Functions backend for a peer-to-peer rental marketplace. It handles booking lifecycle operations, payments, payouts, QR handover/return verification, notifications, recommendation events, admin workflows, and AI-powered listing moderation using Genkit and Gemini.

This backend was developed with AI-assisted engineering workflows using OpenAI Codex, especially for function design, workflow orchestration, tests, moderation logic, and marketplace automation.
